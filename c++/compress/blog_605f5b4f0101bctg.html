CC 与 CXX: 
    这是 C 与 C++ 编译器命令。默认值一般是 “gcc” 与 “g++”。
CPPFLAGS will be given to the C preprocessor
    这是用于预处理阶段的选项。
CFLAGS 与 CXXFLAGS
    CFLAGS 表示用于 C 编译器的选项，CXXFLAGS 表示用于 C++ 编译器的选项。这两个变量实际上涵盖了编译和汇编两个步骤。
